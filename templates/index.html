Esri's ArcGIS Desktop Student Edition and ArcGIS Pro software are available to students, instructors, faculty, and other educators affiliated with Carleton University for educational, research, and academic purposes only. This software is not for commercial use, resale, transfer or exchange. This license is held by an individual person rather than the educational institution.

ArcGIS 10.7 is available for Student One-Year Licenses* with the following eligible extensions: 3D Analyst, Data Interoperability, Geostatistical Analyst, Network Analyst, Publisher, Schematics, Spatial Analyst and Tracking Analyst.

ArcGIS Pro 2.3 is available for student use with the following eligible extensions: 3D Analyst, Data Interoperability, Data Reviewer, Geostatistical Analyst, Network Analyst, Spatial Analyst and Workflow Manager.

*Student One-Year License: A one-year license (365-day evaluation) commencing on registration of the software.

Note: Esri does not offer direct technical support for the ArcGIS Desktop for Student edition and you are not to contact them directly or via the Esri support forums for help with technical issues with installing the software. Please contact GIS@carleton.ca for technical issues.

Questions pertaining to the "hands-on" use of this software should be directed to your course instructor.
